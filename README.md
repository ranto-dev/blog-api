# API RESTful de Blog Posts (MERN Stack)

## À Propos

Ce dépôt contient le **Back-end** d'une plateforme de blog, entièrement développé sur la **stack MERN** (MongoDB, Express, Node.js). L'API est conçue pour gérer le **CRUD** (Création, Lecture, Mise à jour, Suppression) des articles de blog, en intégrant un système d'**authentification robuste via JSON Web Tokens (JWT)**. Il fournit la fondation de données sécurisée et performante pour n'importe quel client Front-end.

## Technologies Utilisées

| Technologie              | Rôle Principal                                                    |
| :----------------------- | :---------------------------------------------------------------- |
| **Node.js**              | Environnement d'exécution côté serveur.                           |
| **Express.js**           | Framework web minimaliste et flexible pour l'API.                 |
| **MongoDB**              | Base de données NoSQL pour le stockage des posts et utilisateurs. |
| **Mongoose**             | Modélisation des données et interaction avec MongoDB.             |
| **JSON Web Token (JWT)** | Gestion de l'authentification et de l'autorisation.               |
| **Bcrypt**               | Hachage sécurisé des mots de passe.                               |
